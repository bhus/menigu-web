BaseConfig

open class BaseConfig

A base class for defining application routes.

Provides a simple DSL for registering routes within the application. User-defined configuration classes should inherit from this class.

Example usage in app/controllers/Config.kt:

class Config : BaseConfig() {
    fun initRoutes() {
        get("/", "Home")
        get("/posts", "Posts", "index")
    }
}
