FrameworkConfig

object FrameworkConfig

A singleton object for managing framework and application configuration.

It loads properties from a settings.properties file. The path to this file can be overridden via the "kahawi.config.path" system property, or by passing a direct path to loadConfig(). This is particularly useful for tests.
